      Follow these steps to start fetching tweets with hawkx
    </p>
  </div>

  <Steps>
    <Step title="Install hawkx">
      Install hawkx using pipx or pip. Make sure you have Python 3.10 or higher installed.

      <CodeGroup>
      ```bash pipx
      pipx install hawkx
      ```

      ```bash pip
      pip install --user hawkx
      ```
      </CodeGroup>

      <Info>pipx is recommended as it installs the CLI in an isolated environment</Info>
    </Step>

    <Step title="Log into X in Chrome">
      hawkx uses your Chrome session cookies to authenticate. Make sure you're logged into x.com in Chrome before running any commands.

      <Note>Currently, hawkx only supports macOS as it uses Keychain to decrypt Chrome cookies</Note>
    </Step>

    <Step title="Fetch your first tweets">
      Run your first command to fetch recent tweets from any user:

      ```bash
      hawkx getuser elonmusk 20
      ```

      This will fetch the 20 most recent tweets from @elonmusk and output them as JSON.

      <Accordion title="Example output">
      ```json
      [
        {
          "id": "1234567890",
          "text": "Just launched our newest product!",
          "created_at": "Mon Mar 04 12:00:00 +0000 2026",
          "user": "elonmusk",
          "favorites": 50000,
          "retweets": 12000,
          "replies": 3400
        }
      ]
      ```
      </Accordion>
    </Step>

    <Step title="Explore more features">
      hawkx supports multiple commands for different use cases:
      
      - Fetch tweets with images using the `-I` flag
      - Get your bookmarks with `hawkx getbookmarks`
      - Retrieve full tweet threads with replies using `-R`
      
      Check out the [commands reference](/commands/getuser) for all available options.
    </Step>
  </Steps>
</div>
